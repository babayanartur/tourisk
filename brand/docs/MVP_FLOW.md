# TOURISK MVP FLOW

Version 1.0

---

## Goal

Build a working Yerevan MVP that makes exploration feel real, visual, and rewarding.

The MVP should prove one thing:

A player can walk through the real world and feel that they are uncovering their own map.

---

## Current Prototype

Tourisk already has a working prototype:

- live map
- user location
- check-ins
- duplicate XP prevention
- visited cells
- green explored map areas
- Firebase profile saving
- early fog / territory logic

This MVP should improve what already works, not restart from zero.

---

## Player Flow

1. Open Tourisk
2. See the map
3. See unexplored areas
4. Walk in the real world
5. Reveal territory
6. Make check-in
7. Earn XP
8. See progress
9. Open profile
10. Feel ownership of explored places
11. Want to explore again

---

## MVP Area

First MVP city:

Yerevan.

Not Armenia.

Not the world.

One city first.

If Yerevan feels good, we expand.

---

## Core MVP Features

- Map
- User location
- Fog of War
- Explored territory
- Check-in
- XP
- Profile
- City progress
- Simple achievements

---

## Success Criteria

The MVP is successful if a player can say:

"I want to walk a little more because I want to open that area."

That is the heart of Tourisk MVP.
