# Tourisk Backend 1.3.0

- Админка управляет достижениями, фигурками и местами.
- Изображения фигурок загружаются в `uploads/pawns`.
- Создание фигурки без изображения запрещено.
- Новые стандартные фигурки имеют прозрачный фон без серых пятен.
- Версия seed-контента увеличена до 2.
- Старые стандартные пути изображений мигрируют на файлы `_v13` для сброса мобильного кэша.
- Пользовательские изображения, загруженные администратором, миграция не трогает.
- Leaderboard возвращает реальные XP, страну, город, выбранную фигурку и дату активности пользователя.
