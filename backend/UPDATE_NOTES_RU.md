# Tourisk Backend — Legendary Discovery MVP

- добавлены 10 легендарных мест Еревана, каждое с наградой `50 XP`;
- изображения автоматически копируются из `seed-assets/places` в `uploads/places` при запуске;
- в модель места добавлены история, иконка и радиус открытия;
- старые ID мест безопасно мигрируются на канонические ID;
- устаревшие тестовые места выключаются, но не удаляются из базы;
- открытые места пользователей мигрируются без потери прогресса;
- начисление XP за место сделано атомарным: повторные и параллельные запросы не дают награду второй раз;
- в публичный конфиг добавлен `dailyJourneyHour: 18`;
- админка позволяет редактировать текст карточки, иконку и радиус открытия места;
- проверка `npm run check` расширена на маршруты открытия, модели и стартовый контент.
