# Tourisk Mobile 1.7 — Home, игровой туман и профиль исследователя

## Home

- Сохранён живой дышащий фон.
- Заголовок «Мир ждёт тебя» остаётся постоянно видимым.
- Добавлена фраза «Каждый шаг открывает твою историю.» с циклом fade in → пауза → fade out.
- Цвет и свечение фразы совпадают с золотым следом на карте.
- Убраны XP-карточка, CTA, статистический блок и вечерняя карточка: Home теперь состоит только из сцены, текста, фигурки и нижней навигации.

## Авторизация

- Активная авторизованная сессия теперь всегда открывает главный экран.
- Intro показывается только новому устройству без активного пользователя.
- После регистрации пользователь больше не возвращается на приветственный экран.

## Карта

- Добавлены три новые ночные объёмные текстуры облаков.
- Один native marker теперь рисует целый облачный кластер из основной массы и второго слоя.
- Облака стали крупнее, темнее и плотнее, но количество marker-компонентов ограничено для защиты слабых Android-устройств.
- Края открытой территории частично перекрываются мягкими облаками, поэтому раскрытие выглядит естественно, а не как геометрическое отверстие.
- Сохранение открытых территорий и отображение фигурки поверх тумана сохранены.

## Профиль

Приоритет данных изменён:

1. Награды.
2. Открытия.
3. Серия дней.
4. Реальные шаги.
5. XP.

- Добавлен блок «История исследователя».
- Достижения перенесены выше последних открытий.
- XP визуально понижен до вторичной метрики.
- Количество шагов вычисляется из реально сохранённой GPS-дистанции со средней длиной шага 0,75 м.
