# Tourisk Mobile 1.3.0

## Что реализовано

### Главная
- Интерфейс собран по финальному макету из `docs/design-v1.3/home-reference.png`.
- Фигурка статична и не имеет серой подложки.
- Убрана мигающая декоративная кнопка за фигуркой.
- XP, открытия, расстояние и достижения берутся из состояния пользователя.
- Нижний информационный блок и навигация подняты с учётом safe area.

### Карта
- Стартовая резервная точка симулятора: Бишкек, Кыргызстан.
- На реальном устройстве используется фактическая GPS-позиция.
- На iOS используется спутниковая карта без названий улиц.
- На Google Maps дополнительно скрыты все подписи стилем карты.
- Туман рисуется одним полигоном с органическими отверстиями, поэтому нет полос между ячейками.
- Добавлены два слоя живых облаков с медленным движением.
- Открытая область формируется вокруг текущей позиции и реально посещённых ячеек.
- Фигурка анимируется только на карте.

### Профиль
- Блок географии перенесён вверх экрана.
- Карта Кыргызстана отрисовывается настоящим SVG-компонентом.
- GPS-точки из реальных check-in отображаются на карте и соединяются маршрутом.
- Нет тестовых открытий и выдуманных городов.

### Лидеры
- Удалены тестовые Artur, Liana, Dmitry и другие демонстрационные пользователи.
- Рейтинг строится только из `/api/leaderboard` и текущего аккаунта.
- Текущий пользователь выделяется отдельной строкой.
- Фильтры используют реальные XP, страны и дату активности.

### Фигурки и достижения
- Фигурки и их изображения управляются через backend/admin.
- Стандартные изображения фигурок лежат в `uploads/pawns` на backend.
- Для версии 1.3 используются новые имена файлов `_v13`, чтобы iOS не показывал старые изображения из кэша.
- При старте backend автоматически мигрирует только старые стандартные пути. Загруженные администратором изображения не перезаписываются.
